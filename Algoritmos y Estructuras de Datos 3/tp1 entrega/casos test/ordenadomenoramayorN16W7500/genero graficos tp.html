      ## Correlacin
      Una manera ms precisa de ver si dos series de datos o variables correlacionan, es intentar ver el crecimiento de las dos series en paralelo. Por ejemplo, por cada valor de n, existe un valor IS(n) que es el tiempo de ejecucin real del algoritmo y un valor cota(n)=n^2. Podemos tomar estos valores ordenados en dos vectores X=[IS(1), IS(2), ...], Y=[cota(1), cota(2), ...] y graficar una serie de puntos.

      ### Coeficiente de correlacin de Pearson
      Para ver si dos funciones correlacionan, es decir, si se parecen y cunto se parecen, podemos usar alguna tcnica un poco ms avanzada. Por ejemplo, el coeficiente de correlacin de Pearson ( https://es.wikipedia.org/wiki/Coeficiente_de_correlacin_de_Pearson ). <br />
      La idea es tomar alguna medida que sea independiente de las magnitudes y solamente indique si dos funciones o series de datos crecen o decrecen del mismo modo (o si hay una correlacin negativa, es decir, cuando uno crece el otro decrece). <br />

      donde xi es el valor de la muestra i de la primera serie de datos, e y_i es el valor de la muestra i de la segunda serie de datos. <br /> <br />
      El resultado de esta expresin es un nmero entre -1 y 1 que indica lo siguiente ( https://es.wikipedia.org/wiki/Coeficiente_de_correlacin_de_Pearson ):
      * Si r = 1, existe una correlacin positiva perfecta. El ndice indica una dependencia total entre las dos variables denominada relacin directa: cuando una de ellas aumenta, la otra tambin lo hace en proporcin constante.
      * Si 0 < r < 1, existe una correlacin positiva.
      * Si r = 0, no existe relacin lineal. Pero esto no necesariamente implica que las variables son independientes: pueden existir todava relaciones no lineales entre las dos variables.
      * Si -1 < r < 0, existe una correlacin negativa.
      * Si r = -1, existe una correlacin negativa perfecta. El ndice indica una dependencia total entre las dos variables llamada relacin inversa: cuando una de ellas aumenta, la otra disminuye en proporcin constante.
